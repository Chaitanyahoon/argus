## üõ†Ô∏è DEPLOYMENT (Now Simplified)
Argus has evolved to be **Fully File-Based**. No MongoDB. No SQLite. Just pure, portable JSON.

### 1. CLONE
```bash
git clone https://github.com/Chaitanyahoon/argus.git
cd argus
```

### 2. INITIALIZE
```bash
npm install
```

### 3. CONFIGURE
Create a `.env` file:
```env
TOKEN=your_discord_bot_token
CLIENT_ID=your_application_id
GEMINI_API_KEY=your_gemini_key
```

### 4. AWAKEN
```bash
npm start
```
*Your data (profiles, XP, settings) will be automatically stored in the `data/` directory.*
